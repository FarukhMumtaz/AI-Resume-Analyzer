# ✨ Overview

AI Resume Analyzer is a modern AI-powered platform that evaluates resumes against target job roles using ATS-inspired scoring and intelligent analysis.

The project was designed and developed under strict hackathon time constraints (≈ 1 hour 15 minutes) by a 3-member team during the **AI Project Making Competition — HITEC University Taxila Cantt (May 2026).**

The system provides intelligent career insights including:

* 📊 ATS Score Estimation
* 🎯 Job Match Percentage
* 🧠 Skill Extraction
* ⚠️ Missing Skills Detection
* 💬 AI Feedback & Recommendations
* ❓ Interview Question Generation
* 📄 Resume Preview & Downloadable Report
* ☁️ Streamlit Cloud Deployment

# ⚡ Installation

## Clone Repository

```bash
git clone https://github.com/FarukhMumtaz/AI-Resume-Analyzer.git
cd AI-Resume-Analyzer
```

## Create Virtual Environment

```bash
python3 -m venv venv
source venv/bin/activate
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run Application

```bash
streamlit run app.py
```
